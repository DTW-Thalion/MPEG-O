.. py:class:: ProvenanceRecord

   A single provenance step describing how data was produced or
   transformed. W3C PROV-compatible.

   :param timestamp_unix: Unix timestamp (seconds since 1970-01-01T00:00:00Z).
   :type timestamp_unix: int, default 0
   :param software: Software name + version (e.g. ``"MSConvert 3.0.24052"``).
   :type software: str, default ""
   :param parameters: Software-specific processing parameters.
   :type parameters: dict[str, Any], default {}
   :param input_refs: URIs / identifiers of input entities.
   :type input_refs: list[str], default []
   :param output_refs: URIs / identifiers of output entities.
   :type output_refs: list[str], default []

   .. rubric:: Notes

   API status: Stable.
