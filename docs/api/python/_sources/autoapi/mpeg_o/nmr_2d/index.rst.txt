.. py:class:: NMR2DSpectrum

   Bases: :py:obj:`mpeg_o.spectrum.Spectrum`


   2-D NMR spectrum: rank-2 intensity matrix with F1 and F2 axes.

   Subclass of :class:`Spectrum`. The matrix is stored as a rank-2
   ``numpy.ndarray`` of shape ``(height, width)``.

   :param intensity_matrix: Rank-2 intensity matrix; shape ``(height, width)``.
   :type intensity_matrix: numpy.ndarray
   :param f1_axis: F1 axis descriptor.
   :type f1_axis: AxisDescriptor or None, default None
   :param f2_axis: F2 axis descriptor.
   :type f2_axis: AxisDescriptor or None, default None
   :param nucleus_f1: Nucleus type on F1 (``"1H"``, ``"13C"``, ...).
   :type nucleus_f1: str, default ""
   :param nucleus_f2: Nucleus type on F2.
   :type nucleus_f2: str, default ""
   :param \*plus all base :class:`Spectrum` parameters*:

   API status: Stable.
