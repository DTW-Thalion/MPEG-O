.. py:class:: InstrumentConfig

   Instrument configuration block persisted as HDF5 string attributes.

   All fields default to empty strings because the format spec
   explicitly allows every sub-field to be missing (§3 of
   ``docs/format-spec.md``).

   :param manufacturer: Instrument manufacturer (e.g. ``"Thermo Fisher Scientific"``).
   :type manufacturer: str, default ""
   :param model: Instrument model (e.g. ``"Orbitrap Eclipse"``).
   :type model: str, default ""
   :param serial_number: Serial number.
   :type serial_number: str, default ""
   :param source_type: Ionization source (e.g. ``"ESI"``, ``"MALDI"``).
   :type source_type: str, default ""
   :param analyzer_type: Mass analyzer (e.g. ``"Orbitrap"``, ``"TOF"``).
   :type analyzer_type: str, default ""
   :param detector_type: Detector (e.g. ``"inductive"``, ``"electron multiplier"``).
   :type detector_type: str, default ""

   .. rubric:: Notes

   API status: Stable.

   ``global.thalion.ttio.InstrumentConfig``.
