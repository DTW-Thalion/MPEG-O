.. py:class:: IRImage

   Infrared hyperspectral imaging dataset: a ``width × height`` grid
   of pixels, each pixel an IR spectrum of ``spectral_points`` intensity
   values sampled at a shared rank-1 ``wavenumbers`` axis.

   :param width: Image grid dimensions in pixels.
   :type width: int, default 0
   :param height: Image grid dimensions in pixels.
   :type height: int, default 0
   :param spectral_points: Number of float64 intensity values per pixel.
   :type spectral_points: int, default 0
   :param intensity: Rank-3 intensity cube of shape ``(height, width, spectral_points)``.
   :type intensity: numpy.ndarray, default empty rank-3 array
   :param wavenumbers: Rank-1 wavenumber axis of length ``spectral_points`` (1/cm).
   :type wavenumbers: numpy.ndarray, default empty rank-1 array
   :param pixel_size_x: Spatial pixel size (implementation-defined units).
   :type pixel_size_x: float, default 0.0
   :param pixel_size_y: Spatial pixel size (implementation-defined units).
   :type pixel_size_y: float, default 0.0
   :param scan_pattern: Scan pattern label (``"raster"``, ``"random_access"``, ...).
   :type scan_pattern: str, default ""
   :param tile_size: HDF5 chunk tile size for reads; 0 means native cube chunks.
   :type tile_size: int, default 0
   :param mode: Whether y-values are absorbance or transmittance.
   :type mode: IRMode, default ``IRMode.TRANSMITTANCE``
   :param resolution_cm_inv: Spectral resolution in reciprocal centimetres.
   :type resolution_cm_inv: float, default 0.0
   :param title: Dataset-level metadata.
   :type title: str, default ""
   :param isa_investigation_id: Dataset-level metadata.
   :type isa_investigation_id: str, default ""
   :param identifications: Dataset-level composition fields.
   :type identifications: list, default []
   :param quantifications: Dataset-level composition fields.
   :type quantifications: list, default []
   :param provenance_records: Dataset-level composition fields.
   :type provenance_records: list, default []
