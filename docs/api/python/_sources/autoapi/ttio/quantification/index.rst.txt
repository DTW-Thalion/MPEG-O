.. py:class:: Quantification

   An abundance observation for a chemical entity in a sample.

   :param chemical_entity: CHEBI accession or chemical formula.
   :type chemical_entity: str
   :param sample_ref: Sample identifier.
   :type sample_ref: str
   :param abundance: Measured abundance.
   :type abundance: float
   :param normalization_method: Normalization method. Empty string represents an unnormalized
                                abundance.
   :type normalization_method: str, default ""

   .. rubric:: Notes

   API status: Stable.
