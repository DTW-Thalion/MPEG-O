.. py:class:: UVVisSpectrum

   Bases: :py:obj:`ttio.spectrum.Spectrum`


   A 1-D UV-Vis spectrum with ``wavelength`` (nm) and ``absorbance`` channels.

   Unlike Raman / IR spectra, UV-Vis observations are indexed by
   wavelength in nanometres rather than wavenumber.

   :param path_length_cm: Optical path length of the cuvette in centimetres.
   :type path_length_cm: float, default 0.0
   :param solvent: Free-form solvent description (e.g. ``"methanol"``).
   :type solvent: str, default ``""``
   :param \*plus all base :class:`Spectrum` parameters*:
