.. py:class:: AxisDescriptor

   Describes a single axis of a :class:`~ttio.signal_array.SignalArray`.

   Immutable value class. ``name`` is a short token (``"mz"``,
   ``"chemical_shift"``, ...) and ``unit`` is a free-form label
   (``"m/z"``, ``"ppm"``, ``"s"``, ...).

   :param name: Semantic name of the axis.
   :type name: str
   :param unit: Unit of the axis values.
   :type unit: str
   :param value_range: Numeric bounds of the axis. ``None`` when unknown.
   :type value_range: ValueRange or None, default None
   :param sampling_mode: Whether samples are regularly spaced.
   :type sampling_mode: SamplingMode, default SamplingMode.UNIFORM

   .. rubric:: Notes

   API status: Stable.
