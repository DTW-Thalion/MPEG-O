.. py:class:: MassSpectrum

   Bases: :py:obj:`ttio.spectrum.Spectrum`


   A mass spectrum: m/z + intensity arrays plus MS level, polarity,
   and an optional scan window.

   :param ms_level: MS level (1, 2, 3, ...).
   :type ms_level: int, default 1
   :param polarity: Ion polarity.
   :type polarity: Polarity, default Polarity.UNKNOWN
   :param scan_window: m/z range covered by the scan. ``None`` if not reported.
   :type scan_window: ValueRange or None, default None
   :param activation_method: Precursor activation method for MS2+ scans. ``NONE`` for MS1
                             and for MS2 scans whose activation was not reported.
   :type activation_method: ActivationMethod, default ActivationMethod.NONE
   :param isolation_window: Precursor isolation window for MS2+ scans. ``None`` for MS1
                            and when the window was not reported.
   :type isolation_window: IsolationWindow or None, default None
   :param \*plus all base :class:`Spectrum` parameters (signal_arrays:
   :param axes:
   :param :
   :param index_position:
   :param scan_time_seconds:
   :param precursor_mz:
   :param precursor_charge)*:

   .. rubric:: Notes

   API status: Stable.

   Standard channel names are ``"mz"`` and ``"intensity"``.
