   Chromatogram: time-vs-intensity trace. TIC, XIC, or SRM.

   Subclass of :class:`Spectrum`. The time and intensity arrays live
   in :attr:`signal_arrays` under keys ``"time"`` and ``"intensity"``.

   :param chromatogram_type: TIC / XIC / SRM.
   :type chromatogram_type: ChromatogramType, default ChromatogramType.TIC
   :param target_mz: XIC target m/z. ``0`` if not XIC.
   :type target_mz: float, default 0.0
   :param product_mz: SRM product m/z. ``0`` if not SRM.
   :type product_mz: float, default 0.0
   :param \*plus all base :class:`Spectrum` parameters*:

   .. rubric:: Notes

   API status: Stable.

   Standard channel names are ``"time"`` and ``"intensity"``.

   Cross-language equivalents
   --------------------------
   Objective-C: ``TTIOChromatogram`` · Java:
   ``global.thalion.ttio.Chromatogram``.
