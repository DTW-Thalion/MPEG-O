.. py:class:: IRSpectrum

   A 1-D infrared spectrum with ``wavenumber`` and ``intensity`` channels.

   :param mode: Whether y-values are absorbance or transmittance.
   :type mode: IRMode, default ``IRMode.TRANSMITTANCE``
   :param resolution_cm_inv: Spectral resolution in reciprocal centimetres.
   :type resolution_cm_inv: float, default 0.0
   :param number_of_scans: Number of co-added scans producing this spectrum.
   :type number_of_scans: int, default 0
   :param \*plus all base :class:`Spectrum` parameters*:
