ttio.exporters.nmrml
====================

.. py:module:: ttio.exporters.nmrml

.. autoapi-nested-parse::

   nmrML writer — Milestone 29.

   Serializes an :class:`ttio.NMRSpectrum` (and optionally its FID) to
   an nmrML XML document. Output mirrors the elements parsed by
   ``ttio.importers.nmrml`` so it round-trips through the reader.

   SPDX-License-Identifier: Apache-2.0

   Cross-language equivalents
   --------------------------
   Objective-C: ``TTIONmrMLWriter`` · Java:
   ``global.thalion.ttio.exporters.NmrMLWriter``

   API status: Stable.
