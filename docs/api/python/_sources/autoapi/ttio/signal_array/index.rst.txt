.. py:class:: SignalArray

   A named 1-D numeric signal buffer paired with axis and encoding
   metadata, with CV annotation support.

   :param data: 1-D numeric array holding the raw signal values. Callers should
                treat the array contents as immutable after construction.
   :type data: numpy.ndarray
   :param axis: Descriptor of the physical dimension represented by ``data``
                (e.g. m/z, ppm). ``None`` when the axis is not yet known or not
                applicable.
   :type axis: AxisDescriptor or None, optional
   :param encoding: Wire-format encoding for HDF5 serialisation.  Defaults to
                    FLOAT64 / ZLIB / little-endian.
   :type encoding: EncodingSpec, optional
   :param cv_params: Initial CV annotations.  Usually empty; grow at runtime via
                     :meth:`add_cv_param`.
   :type cv_params: list of CVParam, optional

   .. rubric:: Notes

   API status: Stable.

   Cross-language equivalents
   --------------------------
   Objective-C: ``TTIOSignalArray`` ·
   Java: ``global.thalion.ttio.SignalArray``
