.. py:class:: CVParam

   A single controlled-vocabulary parameter reference.

   Immutable value class in the form
   ``(ontology_ref, accession, name, [value], [unit])``.

   :param ontology_ref: Ontology short name (``"MS"``, ``"NMR"``, ``"UO"``, ...).
   :type ontology_ref: str
   :param accession: Ontology accession in ``<CV>:<id>`` form
                     (e.g. ``"MS:1000515"``).
   :type accession: str
   :param name: Human-readable label.
   :type name: str
   :param value: Optional free-form value. Use an empty string when the
                 parameter is a pure assertion with no associated value.
   :type value: str, default ""
   :param unit: Optional unit accession (typically a UO or MS term).
   :type unit: str or None, default None

   .. rubric:: Notes

   API status: Stable.
