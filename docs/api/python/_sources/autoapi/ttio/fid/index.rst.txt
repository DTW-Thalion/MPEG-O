   NMR free-induction-decay signal.

   Subclass of :class:`SignalArray` holding a complex-valued buffer
   plus FID-specific acquisition metadata: dwell time, scan count,
   receiver gain.

   :param dwell_time_seconds: Sample spacing in seconds.
   :type dwell_time_seconds: float, default 0.0
   :param scan_count: Number of scans averaged.
   :type scan_count: int, default 1
   :param receiver_gain: Receiver gain used during acquisition.
   :type receiver_gain: float, default 1.0
   :param \*plus all base :class:`SignalArray` parameters (data:
   :param axis:

   :param encoding:
   :param cv_params)*:
