   2D correlation spectrum (Noda 2D-COS): synchronous + asynchronous matrices.

   A 2D-COS observation carries a pair of square rank-2 correlation
   matrices — synchronous (in-phase) and asynchronous
   (quadrature) — both keyed on the same spectral-variable axis
   (e.g. wavenumber ν₁ = ν₂). The diagonal of the synchronous matrix
   is the perturbation-modulated autocorrelation; off-diagonal
   cross-peaks report simultaneous changes. Off-diagonal asynchronous
   cross-peaks report sequential changes.

   :param synchronous_matrix: Rank-2 synchronous correlation matrix, shape ``(n, n)``.
   :type synchronous_matrix: numpy.ndarray
   :param asynchronous_matrix: Rank-2 asynchronous correlation matrix, shape ``(n, n)``.
   :type asynchronous_matrix: numpy.ndarray
   :param variable_axis: Shared axis descriptor (applies to both F1 and F2).
   :type variable_axis: AxisDescriptor or None, default None
   :param perturbation: Free-form description of the driving perturbation
                        (``"temperature"``, ``"concentration"``, ...).
   :type perturbation: str, default ""
   :param perturbation_unit: Unit for the perturbation (e.g. ``"K"``, ``"mM"``).
   :type perturbation_unit: str, default ""
   :param source_modality: Originating spectroscopy (``"raman"``, ``"ir"``, ``"uv-vis"``, ...).
   :type source_modality: str, default ""
   :param \*plus all base :class:`Spectrum` parameters*:
