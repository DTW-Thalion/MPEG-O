ttio.transport¶

TTI-O streaming transport codec (v0.10 M67).

Implements the wire format specified in docs/transport-spec.md:

  • PacketHeader — 24-byte fixed header with CRC-32C checksum support.

  • PacketType — enum of 9 packet types (stream/dataset header, access unit, protection metadata, annotation, provenance, chromatogram, end-of-dataset, end-of-stream).

  • AccessUnit — one spectrum’s filter keys + channel bytes.

  • TransportWriter / TransportReader — file and in-memory stream codecs.

  • file_to_transport() / transport_to_file() — convenience one-shot round-trip helpers.

Cross-language: ObjC objc/Source/Transport/ · Java global.thalion.ttio.transport.

class ttio.transport.AUFilter[source]¶

Filter predicate evaluated against an AccessUnit.

All fields are optional; an empty filter accepts every AU. See docs/transport-spec.md §7.

The filter is applied BEFORE an AU is serialized to the wire, so filtered streams consume no bandwidth for non-matching spectra.
