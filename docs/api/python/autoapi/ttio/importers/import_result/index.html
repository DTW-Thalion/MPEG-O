ttio.importers.import_result¶

ImportResult — lightweight in-memory container produced by importers.

The main SpectralDataset class wraps an open HDF5 file, which would be awkward to construct from an importer that has no backing file yet. Instead, importers produce an ImportResult that can be inspected in memory and then flushed to a real .tio file with ImportResult.to_ttio().

Notes

API status: Stable (Python-idiomatic helper; ObjC and Java use inline constructs).
