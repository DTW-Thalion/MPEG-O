ttio.importers.fasta¶

FASTA importer.

Parses FASTA files into either a ReferenceImport (for reference genomes that pair with BAM/CRAM input) or an unaligned WrittenGenomicRun (for amplicon panels, target lists, or quality-stripped reads).

Supports gzip-compressed input transparently (detected via magic bytes 1f 8b). FASTA records are header-line >name [desc...] followed by one or more sequence lines until the next header or EOF.

class ttio.importers.fasta.FastaReader(path: str | os.PathLike[str])[source]¶

Read a FASTA file into TTI-O containers.

Parameters:

path (str or Path) – Filesystem path to a .fa / .fasta / .fna file. gzip-compressed inputs (.fa.gz etc.) are auto-detected and decompressed.

Notes

Construction reads only the file header (or nothing). Calling read_reference() or read_unaligned() triggers the full parse.

read_reference(uri: str | None = None) → ttio.genomic.reference_import.ReferenceImport[source]¶

Parse the file as a reference genome.

Each FASTA record becomes one chromosome.

Parameters:

uri (str, optional) – Reference URI to record on the resulting ReferenceImport. Defaults to the file’s stem (e.g. "GRCh38" for GRCh38.fa), with .fa.gz and .fasta.gz compound suffixes stripped.

Returns:

ReferenceImport – Value class carrying chromosomes, sequences (case- preserving), and content MD5.

read_unaligned(*, sample_name: str = '', platform: str = '', reference_uri: str = '', acquisition_mode: ttio.enums.AcquisitionMode = AcquisitionMode.GENOMIC_WGS) → ttio.written_genomic_run.WrittenGenomicRun[source]¶

Parse the file as a set of unaligned reads.

Each FASTA record becomes one read with SAM-unmapped sentinel values: flags = 4 (unmapped), chromosome = "*", position = 0, mapq = 255, cigar = "*". The qualities channel is filled with 0xff bytes (one per base, matching BamReader’s convention for SAM records whose QUAL field is *).

Parameters:

  • sample_name (str) – Recorded on the genomic run; default "".

  • platform (str) – Sequencing platform tag (e.g. "ILLUMINA"); default "".

  • reference_uri (str) – Reference URI to associate with this run; default "" (no reference).

  • acquisition_mode (AcquisitionMode) – Run-level acquisition mode; default GENOMIC_WGS.

Returns:

WrittenGenomicRun – Run container ready to pass to SpectralDataset.write_minimal() via genomic_runs=[...].
