Software documentation for the MPGOVerifier class

MPGOVerifier : NSObject

Declared in:
MPGOVerifier.h
Higher-level verification API. Collapses the three outcomes of a sign-and-verify cycle (valid / invalid / not-signed) into a single enum, plus an error fallback for I/O failures. Use this instead of MPGOSignatureManager when you want to render a status to an end user. API status: Stable. Cross-language equivalents: Python: mpeg_o.verifier.Verifier Java: com.dtwthalion.mpgo.protection.Verifier
