MPGOFeatureFlags : NSObject

Declared in:
MPGOFeatureFlags.h
Versioning and feature-flag utility for MPGO file format v0.2+. Every file written by v0.2+ carries two root-group attributes: @mpeg_o_format_version = "1.1" (major.minor string) @mpeg_o_features = JSON array of feature strings v0.1 files have @mpeg_o_version = "1.0.0" instead and no @mpeg_o_features — readers detect the old format by the absence of the feature-flags attribute and fall back to the v0.1 JSON paths. Features without an `opt_` prefix are required: a reader must refuse a file that names a required feature it does not support. Features with `opt_` are informational — readers may skip them.
