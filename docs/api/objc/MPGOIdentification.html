MPGOIdentification : NSObject

Declared in:
MPGOIdentification.h
Links a spectrum (by its 0-based index within an acquisition run) to a chemical entity identification, with a confidence score and an evidence chain represented as an ordered list of free-form strings (typically CV accession references such as "MS:1002217 search engine specific score"). API status: Stable. Cross-language equivalents: Python: mpeg_o.identification.Identification Java: com.dtwthalion.mpgo.Identification
