MPGOSpectrum : NSObject

Declared in:
MPGOSpectrum.h
Base class for any spectrum. Holds an ordered dictionary of named MPGOSignalArrays plus the coordinate axes that index them, the spectrum's position in its parent run, scan time, and optional precursor info for tandem MS. Concrete subclasses (MPGOMassSpectrum, MPGONMRSpectrum,...) add their own typed metadata and validation. HDF5 representation: each spectrum is an HDF5 group whose immediate children are MPGOSignalArray sub-groups (one per named array) plus scalar attributes for the metadata fields. API status: Stable. Cross-language equivalents: Python: mpeg_o.spectrum.Spectrum Java: com.dtwthalion.mpgo.Spectrum
