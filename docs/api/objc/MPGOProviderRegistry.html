MPGOProviderRegistry : NSObject

MPGOProviderRegistry.h
Provider registry singleton. Providers self-register via +load . Callers resolve by explicit name or URL scheme. API status: Stable (Provisional per M39 — may change before v1.0). Cross-language equivalents: Python: mpeg_o.providers — module-level functions (discover_providers, open_provider, register_provider) — idiomatic for Python packaging.
