MPGOKeyRotationManager : NSObject

MPGOKeyRotationManager.h
Milestone 25 — envelope encryption + key rotation. Envelope model: a Data Encryption Key (DEK, 32 bytes) encrypts the signal data. A Key Encryption Key (KEK, 32 bytes) wraps the DEK with AES-256-GCM. Rotation re-wraps the DEK under a new KEK without touching any signal dataset, so it's O(1) in file size.
