Software documentation for the MPGOHDF5Dataset class

MPGOHDF5Dataset : NSObject

Declared in:
MPGOHDF5Dataset.h
Thin wrapper around a 1-D HDF5 dataset. Owns its dataset id and (for compound types) the type id; both are released in -dealloc . Datasets are created with a definite element count and precision; the shape cannot be resized after creation.
