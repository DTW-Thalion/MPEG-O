MPGOFreeInductionDecay : MPGOSignalArray

Declared in:
MPGOFreeInductionDecay.h
NMR free-induction decay. Subclass of MPGOSignalArray that uses the Complex128 precision (interleaved real/imag doubles) plus FID-specific acquisition metadata: dwell time, scan count, receiver gain. Length is the number of complex points (i.e. half the number of doubles in the buffer). Cross-language equivalents: Python: mpeg_o.fid.FreeInductionDecay Java: com.dtwthalion.mpgo.FreeInductionDecay
