MPGOStreamable

Declared in:
MPGOStreamable.h
Objects conforming to MPGOStreamable support sequential access with explicit positioning. This enables efficient iteration over large datasets without materializing the entire collection in memory. API status: Stable. Cross-language equivalents: Python: mpeg_o.protocols.Streamable Java: com.dtwthalion.mpgo.protocols.Streamable
