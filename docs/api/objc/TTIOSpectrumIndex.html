Software documentation for the TTIOSpectrumIndex class

TTIOSpectrumIndex : NSObject

Declared in:
TTIOSpectrumIndex.h
Per-spectrum offsets, lengths, and queryable scan metadata for one TTIOAcquisitionRun. Kept as parallel C arrays inside NSData buffers for compact storage and fast iteration. Persisted as five parallel 1-D HDF5 datasets under the run's `spectrum_index/` sub-group plus a sixth one for precursor charge. Range queries (RT, ms_level, polarity) operate on the in-memory arrays and do not touch the signal channels — this is the "compressed-domain query" property of the MPEG-G access-unit model. API status: Stable. Cross-language equivalents: Python: ttio.acquisition_run.SpectrumIndex Java: com.dtwthalion.ttio.SpectrumIndex
