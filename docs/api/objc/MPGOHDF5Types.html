MPGOHDF5TypeForPrecision

hid_t MPGOHDF5TypeForPrecision(MPGOPrecision precision);
Native HDF5 type id for a given precision. The returned id is owned by the caller for compound types (Complex128) and must be closed with H5Tclose; for primitive types it is a builtin id (H5T_NATIVE_*) which must NOT be closed. Use MPGOHDF5TypeIsBuiltin() to decide which.
