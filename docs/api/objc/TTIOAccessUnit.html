Software documentation for the TTIOAccessUnit class

TTIOAccessUnit : NSObject

Declared in:
TTIOAccessUnit.h
Transport-layer Access Unit: one spectrum as a transport payload. ``spectrumClass`` wire values: 0 = MassSpectrum, 1 = NMRSpectrum, 2 = NMR2D, 3 = FID, 4 = MSImagePixel, 5 = GenomicRead ``polarity`` wire values (differs from TTIOPolarity which uses -1 for negative; the wire uses nonneg only): 0 = positive, 1 = negative, 2 = unknown
