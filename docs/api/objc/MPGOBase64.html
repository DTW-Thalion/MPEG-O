decodeString: 

+ (NSData*) decodeString: (NSString*)base64String;
Decode a base64 string. Whitespace and newlines are tolerated. Returns nil on malformed input.

decodeString: zlibInflate: 

+ (NSData*) decodeString: (NSString*)base64String zlibInflate: (BOOL)inflate;
Decode and optionally zlib-inflate. If inflate is NO, behaves like -decodeString: . If YES, runs uncompress() on the decoded bytes; returns nil if the output is not a valid zlib stream.

encodeData: 

+ (NSString*) encodeData: (NSData*)data;
Base64-encode ``data``. Returns ``@""`` for nil input.

encodeData: zlibDeflate: 

+ (NSString*) encodeData: (NSData*)data zlibDeflate: (BOOL)deflateFlag;
Optionally zlib-compress ``data`` before base64-encoding so the output matches what mzML readers expect for arrays annotated with ``MS:1000574`` (zlib compression).
