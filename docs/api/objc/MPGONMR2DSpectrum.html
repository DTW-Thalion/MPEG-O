MPGONMR2DSpectrum : MPGOSpectrum

Declared in:
MPGONMR2DSpectrum.h
2-D NMR spectrum (e.g. HSQC, COSY): a row-major float64 intensity matrix of `width × height` points plus F1 and F2 axis descriptors. The matrix is stored on disk as a flattened 1-D dataset with the shape recorded in scalar attributes; this is sufficient for round-trip fidelity and keeps the HDF5 wrapper API surface small. A future milestone may switch to a native 2-D dataset.
