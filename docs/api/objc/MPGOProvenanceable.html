MPGOProvenanceable

Declared in:
MPGOProvenanceable.h
Objects conforming to MPGOProvenanceable carry a W3C PROV-compatible chain of processing records. Every transformation applied to the data contributes an entry; the chain makes the object self-documenting and supports regulatory audit trails. API status: Stable.
