Software documentation for the MPGOChromatogram class

MPGOChromatogram : MPGOSpectrum

Declared in:
MPGOChromatogram.h
Chromatogram: time-vs-intensity trace, with TIC, XIC, or SRM type. For XIC the targetMz field is meaningful; for SRM the precursorMz + productMz fields apply (transition); both are zero for TIC. Cross-language equivalents: Python: mpeg_o.chromatogram.Chromatogram Java: com.dtwthalion.mpgo.Chromatogram
