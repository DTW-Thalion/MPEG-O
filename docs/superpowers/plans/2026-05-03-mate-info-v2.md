# mate_info v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `MATE_INLINE_V2 = 13` codec implementing CRAM-style 4-substream encoding (MF / NS / NP / TS) of mate-pair info across native C + Python + Java + ObjC; soft wire-format addition defaulting on at v1.7 with opt-out flag. Target ≥5 MB savings on chr22 NA12878 lean+mapped (goal 7-8 MB).

**Architecture:** Shared C kernel in `native/src/mate_info_v2.{c,h}` mirrors V4 Stage 2 pattern. Each language wraps the C entry points (Python ctypes / Java JNI / ObjC direct link). Substreams use the existing `ttio_rans_encode_block` / `ttio_rans_decode_block` for entropy; no new entropy coder. New on-disk path `signal_channels/mate_info/inline_v2` (uint8 blob, `@compression = 13`) coexists with v1 layout; reader dispatches on path presence; writer dispatches on `opt_disable_inline_mate_info_v2: bool = False` flag.

**Tech Stack:** C11 (kernel), CMake (build), pytest (Python), JUnit + Maven (Java), GNUstep + GNU Make (ObjC), ctypes (Python), JNI (Java).

**Spec:** `docs/superpowers/specs/2026-05-03-mate-info-v2-design.md` (HEAD `fa412db`).

```c
#include "mate_info_v2.h"
#include "../include/ttio_rans.h"

#include <string.h>

/* ── varint (LEB128, little-endian base-128) ───────────────────── */

size_t miv2_varint_encode(uint64_t value, uint8_t *out) {
    size_t i = 0;
    while (value >= 0x80) {
        out[i++] = (uint8_t)(value | 0x80);
        value >>= 7;
    }
    out[i++] = (uint8_t)value;
    return i;
}

int miv2_varint_decode(const uint8_t *in, size_t in_len, uint64_t *out_value, size_t *out_consumed) {
    uint64_t result = 0;
    size_t i = 0;
    int shift = 0;
    while (i < in_len) {
        uint8_t b = in[i++];
        result |= ((uint64_t)(b & 0x7F)) << shift;
        if ((b & 0x80) == 0) {
            *out_value = result;
            *out_consumed = i;
            return 0;
        }
        shift += 7;
        if (shift >= 64) return TTIO_RANS_ERR_CORRUPT;  /* overflow */
    }
    return TTIO_RANS_ERR_CORRUPT;  /* ran out of bytes */
}

/* ── zigzag for int64 ──────────────────────────────────────────── */

uint64_t miv2_zigzag_encode_64(int64_t value) {
    return ((uint64_t)value << 1) ^ (uint64_t)(value >> 63);
}

int64_t miv2_zigzag_decode_64(uint64_t value) {
    return (int64_t)((value >> 1) ^ -(int64_t)(value & 1));
}
```

```c
/* I1: MF exhaustiveness — every (own, mate) pair maps to exactly one MF. */
static void test_i1_mf_exhaustiveness(void) {
    int K = 16;
    for (int own = 0; own < K; own++) {
        for (int mate = -1; mate < K; mate++) {
            int matches = 0;
            if (mate == own)               matches++;  /* MF=0 */
            else if (mate >= 0)            matches++;  /* MF=1 */
            else                           matches++;  /* MF=2 (mate==-1) */
            assert(matches == 1);
        }
    }
    printf("I1 MF exhaustiveness: PASS\n");
}

/* I6: end-to-end lossless round-trip on synthetic mate triples. */
static void test_i6_end_to_end(void) {
    const uint64_t N = 1000;
    int32_t  *mc = malloc(N * sizeof(int32_t));
    int64_t  *mp = malloc(N * sizeof(int64_t));
    int32_t  *ts = malloc(N * sizeof(int32_t));

/* I7: decoder rejects MF=3 in rANS path. */
static void test_i7_reserved_mf_rans_rejection(void) {
    /* Construct a minimal valid v2 blob, then mutate a decoded MF byte
     * to 3 by directly synthesising a 1-record blob with rANS-O0
     * MF stream containing value 3. We'll encode 1 record with MF=2,
     * then patch the MF stream byte to 0x03. */
    int32_t  mc[1] = {-1};
    int64_t  mp[1] = {0};
    int32_t  ts[1] = {0};
    uint16_t oc[1] = {0};
    int64_t  op[1] = {0};

    uint8_t encoded[256];
    size_t encoded_size = sizeof(encoded);
    int rc = ttio_mate_info_v2_encode(mc, mp, ts, oc, op, 1, encoded, &encoded_size);
    assert(rc == 0);

    /* Find MF stream offset (after 34-byte header) and corrupt the
     * 2nd byte (skipping the leading-byte selector at MF[0]).
     * For 1 record, raw-pack will be smaller (1 + ceil(1/4) = 2 bytes),
     * so MF stream is [0x00, 0x?? with low 2 bits = 2]. Patch low 2 bits
     * to 3. */
    encoded[34] = MIV2_MF_RAW_PACK;  /* ensure raw-pack path */
    encoded[35] = 0x03;              /* MF[0] = 3 */
    /* (Note: this also requires MF_LEN = 2 in header. Encode for N=1
     * with all-NO_MATE typically picks raw-pack; verify by reading
     * the header MF_LEN field at offset 22 (4+1+1+8+4 = 18, then
     * MF_LEN starts at 18 = 4+1+1+8+4). */

    int32_t mc2[1]; int64_t mp2[1]; int32_t ts2[1];
    rc = ttio_mate_info_v2_decode(encoded, encoded_size, oc, op, 1, mc2, mp2, ts2);
    assert(rc == TTIO_RANS_ERR_RESERVED_MF);
    printf("I7 reserved MF rejection: PASS\n");
}
```

```c
#include <stdio.h>

/* Forward decls for helpers used internally. */
static int miv2_classify_mf(int32_t mate_chrom_id, uint16_t own_chrom_id, uint8_t *out_mf);
static size_t miv2_encode_mf_raw_pack(const uint8_t *mf, uint64_t n, uint8_t *out);

/* MF classification — central to the taxonomy.
 *  Returns 0 on success, TTIO_RANS_ERR_PARAM on invalid input. */
static int miv2_classify_mf(int32_t mate_chrom_id, uint16_t own_chrom_id, uint8_t *out_mf) {
    if (mate_chrom_id < -1) return TTIO_RANS_ERR_PARAM;
    int32_t own = (own_chrom_id == MIV2_OWN_UNMAPPED) ? -1 : (int32_t)own_chrom_id;
    if (mate_chrom_id == -1) {
        *out_mf = MIV2_MF_NO_MATE;
    } else if (mate_chrom_id == own) {
        *out_mf = MIV2_MF_SAME_CHROM;
    } else {
        *out_mf = MIV2_MF_CROSS_CHROM;
    }
    return 0;
}

/* MF raw-pack encode: 2 bits per MF, 4 per byte, LSB-first. */
static size_t miv2_encode_mf_raw_pack(const uint8_t *mf, uint64_t n, uint8_t *out) {
    size_t bytes = (n + 3) / 4;
    memset(out, 0, bytes);
    for (uint64_t i = 0; i < n; i++) {
        uint8_t v = mf[i] & 0x3;
        size_t byte_idx = i / 4;
        size_t bit_pos = (i % 4) * 2;
        out[byte_idx] |= (uint8_t)(v << bit_pos);
    }
    return bytes;
}

/* MF raw-pack decode: validates each 2-bit slot is in {0,1,2};
 * returns TTIO_RANS_ERR_RESERVED_MF if any slot is 3. */
static int miv2_decode_mf_raw_pack(const uint8_t *in, size_t in_len, uint64_t n, uint8_t *out_mf) {
    size_t bytes = (n + 3) / 4;
    if (in_len < bytes) return TTIO_RANS_ERR_CORRUPT;
    for (uint64_t i = 0; i < n; i++) {
        size_t byte_idx = i / 4;
        size_t bit_pos = (i % 4) * 2;
        uint8_t v = (in[byte_idx] >> bit_pos) & 0x3;
        if (v == MIV2_MF_RESERVED) return TTIO_RANS_ERR_RESERVED_MF;
        out_mf[i] = v;
    }
    return 0;
}

/* Helper: write u32 / u64 LE into buf at offset. */
static void miv2_write_u32_le(uint8_t *buf, uint32_t v) {
    buf[0] = (uint8_t)(v & 0xff);
    buf[1] = (uint8_t)((v >> 8) & 0xff);
    buf[2] = (uint8_t)((v >> 16) & 0xff);
    buf[3] = (uint8_t)((v >> 24) & 0xff);
}
static void miv2_write_u64_le(uint8_t *buf, uint64_t v) {
    for (int i = 0; i < 8; i++) buf[i] = (uint8_t)((v >> (i*8)) & 0xff);
}
static uint32_t miv2_read_u32_le(const uint8_t *buf) {
    return (uint32_t)buf[0] | ((uint32_t)buf[1] << 8) |
           ((uint32_t)buf[2] << 16) | ((uint32_t)buf[3] << 24);
}
static uint64_t miv2_read_u64_le(const uint8_t *buf) {
    uint64_t v = 0;
    for (int i = 0; i < 8; i++) v |= ((uint64_t)buf[i]) << (i*8);
    return v;
}

/* The full encoder. */
int ttio_mate_info_v2_encode(
    const int32_t  *mate_chrom_ids,
    const int64_t  *mate_positions,
    const int32_t  *template_lengths,
    const uint16_t *own_chrom_ids,
    const int64_t  *own_positions,
    uint64_t        n_records,
    uint8_t        *out,
    size_t         *out_len)
{
    if (!mate_chrom_ids || !mate_positions || !template_lengths ||
        !own_chrom_ids || !own_positions || !out || !out_len) {
        return TTIO_RANS_ERR_PARAM;
    }

    /* 1. Classify MF for every record (this also validates inputs). */
    uint8_t *mf = malloc(n_records);
    if (!mf) return TTIO_RANS_ERR_ALLOC;
    uint64_t num_cross = 0;
    for (uint64_t i = 0; i < n_records; i++) {
        int rc = miv2_classify_mf(mate_chrom_ids[i], own_chrom_ids[i], &mf[i]);
        if (rc != 0) { free(mf); return rc; }
        if (mf[i] == MIV2_MF_CROSS_CHROM) num_cross++;
        if (mf[i] == MIV2_MF_RESERVED) { free(mf); return TTIO_RANS_ERR_RESERVED_MF; }
    }

    /* 2. Build NS substream (varint(mate_chrom_id) per CROSS_CHROM record). */
    uint8_t *ns_buf = malloc(num_cross * 10 + 1);
    size_t ns_pos_size = 0;
    if (!ns_buf) { free(mf); return TTIO_RANS_ERR_ALLOC; }
    for (uint64_t i = 0; i < n_records; i++) {
        if (mf[i] == MIV2_MF_CROSS_CHROM) {
            ns_pos_size += miv2_varint_encode((uint64_t)mate_chrom_ids[i], ns_buf + ns_pos_size);
        }
    }

    /* 3. Build NP substream (zigzag_varint per record). */
    uint8_t *np_buf = malloc(n_records * 10 + 1);
    size_t np_pos_size = 0;
    if (!np_buf) { free(mf); free(ns_buf); return TTIO_RANS_ERR_ALLOC; }
    for (uint64_t i = 0; i < n_records; i++) {
        int64_t v;
        if (mf[i] == MIV2_MF_SAME_CHROM) {
            v = mate_positions[i] - own_positions[i];
        } else {
            v = mate_positions[i];
        }
        np_pos_size += miv2_varint_encode(miv2_zigzag_encode_64(v), np_buf + np_pos_size);
    }

    /* 4. Build TS substream (zigzag_varint per record). */
    uint8_t *ts_buf = malloc(n_records * 10 + 1);
    size_t ts_pos_size = 0;
    if (!ts_buf) { free(mf); free(ns_buf); free(np_buf); return TTIO_RANS_ERR_ALLOC; }
    for (uint64_t i = 0; i < n_records; i++) {
        ts_pos_size += miv2_varint_encode(
            miv2_zigzag_encode_64((int64_t)template_lengths[i]),
            ts_buf + ts_pos_size);
    }

    /* 5. Apply rANS-O0 to NS / NP / TS substreams. MF gets auto-pick. */
    /* MF auto-pick: try raw-pack vs rANS-O0, take smaller. */
    size_t mf_raw_size = (n_records + 3) / 4;
    uint8_t *mf_raw = malloc(mf_raw_size + 1);
    if (!mf_raw) goto err_alloc;
    miv2_encode_mf_raw_pack(mf, n_records, mf_raw);

    /* rANS-O0 of MF byte stream. */
    size_t mf_rans_cap = n_records * 2 + 256;
    uint8_t *mf_rans = malloc(mf_rans_cap);
    if (!mf_rans) { free(mf_raw); goto err_alloc; }
    size_t mf_rans_size = mf_rans_cap;
    int rc = ttio_rans_encode_block(mf, n_records, 0 /*order*/, mf_rans, &mf_rans_size);
    if (rc != 0) { free(mf_raw); free(mf_rans); goto err_alloc; }

    int mf_use_rans = (mf_rans_size < mf_raw_size);
    size_t mf_substream_size = 1 + (mf_use_rans ? mf_rans_size : mf_raw_size);

    /* rANS-O0 the other substreams. */
    size_t ns_rans_cap = ns_pos_size * 2 + 256;
    uint8_t *ns_rans = malloc(ns_rans_cap ? ns_rans_cap : 1);
    size_t ns_rans_size = ns_rans_cap;
    if (ns_pos_size > 0) {
        rc = ttio_rans_encode_block(ns_buf, ns_pos_size, 0, ns_rans, &ns_rans_size);
        if (rc != 0) { free(mf_raw); free(mf_rans); free(ns_rans); goto err_alloc; }
    } else {
        ns_rans_size = 0;
    }

    size_t np_rans_cap = np_pos_size * 2 + 256;
    uint8_t *np_rans = malloc(np_rans_cap);
    size_t np_rans_size = np_rans_cap;
    rc = ttio_rans_encode_block(np_buf, np_pos_size, 0, np_rans, &np_rans_size);
    if (rc != 0) { free(mf_raw); free(mf_rans); free(ns_rans); free(np_rans); goto err_alloc; }

    size_t ts_rans_cap = ts_pos_size * 2 + 256;
    uint8_t *ts_rans = malloc(ts_rans_cap);
    size_t ts_rans_size = ts_rans_cap;
    rc = ttio_rans_encode_block(ts_buf, ts_pos_size, 0, ts_rans, &ts_rans_size);
    if (rc != 0) { free(mf_raw); free(mf_rans); free(ns_rans); free(np_rans); free(ts_rans); goto err_alloc; }

    /* 6. Assemble container. */
    size_t total = MIV2_HEADER_SIZE + mf_substream_size + ns_rans_size + np_rans_size + ts_rans_size;
    if (*out_len < total) {
        free(mf_raw); free(mf_rans); free(ns_rans); free(np_rans); free(ts_rans);
        free(mf); free(ns_buf); free(np_buf); free(ts_buf);
        return TTIO_RANS_ERR_PARAM;  /* output buffer too small */
    }

    /* Container header: */
    memcpy(out, MIV2_MAGIC, MIV2_MAGIC_LEN);
    out[4] = MIV2_VERSION;
    out[5] = 0;  /* flags */
    miv2_write_u64_le(out + 6,  n_records);
    miv2_write_u32_le(out + 14, (uint32_t)num_cross);
    miv2_write_u32_le(out + 18, (uint32_t)mf_substream_size);
    miv2_write_u32_le(out + 22, (uint32_t)ns_rans_size);
    miv2_write_u32_le(out + 26, (uint32_t)np_rans_size);
    miv2_write_u32_le(out + 30, (uint32_t)ts_rans_size);

    /* Substreams: */
    size_t off = MIV2_HEADER_SIZE;
    out[off++] = mf_use_rans ? MIV2_MF_RANS_O0 : MIV2_MF_RAW_PACK;
    if (mf_use_rans) { memcpy(out + off, mf_rans, mf_rans_size); off += mf_rans_size; }
    else             { memcpy(out + off, mf_raw,  mf_raw_size);  off += mf_raw_size; }
    memcpy(out + off, ns_rans, ns_rans_size); off += ns_rans_size;
    memcpy(out + off, np_rans, np_rans_size); off += np_rans_size;
    memcpy(out + off, ts_rans, ts_rans_size); off += ts_rans_size;

    *out_len = off;
    free(mf_raw); free(mf_rans); free(ns_rans); free(np_rans); free(ts_rans);
    free(mf); free(ns_buf); free(np_buf); free(ts_buf);
    return 0;

err_alloc:
    free(mf); free(ns_buf); free(np_buf); free(ts_buf);
    return TTIO_RANS_ERR_ALLOC;
}

/* The decoder. */
int ttio_mate_info_v2_decode(
    const uint8_t  *encoded,
    size_t          encoded_size,
    const uint16_t *own_chrom_ids,
    const int64_t  *own_positions,
    uint64_t        n_records,
    int32_t        *out_mate_chrom_ids,
    int64_t        *out_mate_positions,
    int32_t        *out_template_lengths)
{
    if (!encoded || !own_chrom_ids || !own_positions ||
        !out_mate_chrom_ids || !out_mate_positions || !out_template_lengths) {
        return TTIO_RANS_ERR_PARAM;
    }
    if (encoded_size < MIV2_HEADER_SIZE) return TTIO_RANS_ERR_CORRUPT;
    if (memcmp(encoded, MIV2_MAGIC, MIV2_MAGIC_LEN) != 0) return TTIO_RANS_ERR_CORRUPT;
    if (encoded[4] != MIV2_VERSION) return TTIO_RANS_ERR_CORRUPT;
    if (encoded[5] != 0) return TTIO_RANS_ERR_CORRUPT;  /* flags reserved */

    uint64_t hdr_n_records = miv2_read_u64_le(encoded + 6);
    if (hdr_n_records != n_records) return TTIO_RANS_ERR_PARAM;
    uint32_t num_cross  = miv2_read_u32_le(encoded + 14);
    uint32_t mf_len     = miv2_read_u32_le(encoded + 18);
    uint32_t ns_len     = miv2_read_u32_le(encoded + 22);
    uint32_t np_len     = miv2_read_u32_le(encoded + 26);
    uint32_t ts_len     = miv2_read_u32_le(encoded + 30);

    if ((size_t)MIV2_HEADER_SIZE + mf_len + ns_len + np_len + ts_len > encoded_size)
        return TTIO_RANS_ERR_CORRUPT;

    const uint8_t *mf_stream = encoded + MIV2_HEADER_SIZE;
    const uint8_t *ns_stream = mf_stream + mf_len;
    const uint8_t *np_stream = ns_stream + ns_len;
    const uint8_t *ts_stream = np_stream + np_len;

    /* Decode MF stream. */
    uint8_t *mf = malloc(n_records);
    if (!mf) return TTIO_RANS_ERR_ALLOC;
    if (mf_len < 1) { free(mf); return TTIO_RANS_ERR_CORRUPT; }
    int rc;
    if (mf_stream[0] == MIV2_MF_RAW_PACK) {
        rc = miv2_decode_mf_raw_pack(mf_stream + 1, mf_len - 1, n_records, mf);
    } else if (mf_stream[0] == MIV2_MF_RANS_O0) {
        rc = ttio_rans_decode_block(mf_stream + 1, mf_len - 1, mf, n_records);
        if (rc == 0) {
            for (uint64_t i = 0; i < n_records; i++) {
                if (mf[i] == MIV2_MF_RESERVED) { rc = TTIO_RANS_ERR_RESERVED_MF; break; }
                if (mf[i] > MIV2_MF_RESERVED)  { rc = TTIO_RANS_ERR_CORRUPT; break; }
            }
        }
    } else {
        rc = TTIO_RANS_ERR_CORRUPT;
    }
    if (rc != 0) { free(mf); return rc; }

    /* Decode NS stream (rANS → varints). */
    uint8_t *ns_buf = NULL;
    size_t ns_decoded_size = num_cross * 10 + 16;
    if (num_cross > 0) {
        ns_buf = malloc(ns_decoded_size);
        if (!ns_buf) { free(mf); return TTIO_RANS_ERR_ALLOC; }
        rc = ttio_rans_decode_block(ns_stream, ns_len, ns_buf, ns_decoded_size);
        if (rc < 0) { free(mf); free(ns_buf); return rc; }
        ns_decoded_size = (size_t)rc;
    }

    /* Decode NP stream (rANS → varints, n_records of them). */
    size_t np_decoded_cap = n_records * 10 + 16;
    uint8_t *np_buf = malloc(np_decoded_cap);
    if (!np_buf) { free(mf); free(ns_buf); return TTIO_RANS_ERR_ALLOC; }
    rc = ttio_rans_decode_block(np_stream, np_len, np_buf, np_decoded_cap);
    if (rc < 0) { free(mf); free(ns_buf); free(np_buf); return rc; }
    size_t np_decoded_size = (size_t)rc;

    /* Decode TS stream (rANS → varints, n_records of them). */
    size_t ts_decoded_cap = n_records * 10 + 16;
    uint8_t *ts_buf = malloc(ts_decoded_cap);
    if (!ts_buf) { free(mf); free(ns_buf); free(np_buf); return TTIO_RANS_ERR_ALLOC; }
    rc = ttio_rans_decode_block(ts_stream, ts_len, ts_buf, ts_decoded_cap);
    if (rc < 0) { free(mf); free(ns_buf); free(np_buf); free(ts_buf); return rc; }
    size_t ts_decoded_size = (size_t)rc;

    /* Walk the streams, reconstructing tuples. */
    size_t ns_off = 0, np_off = 0, ts_off = 0;
    uint32_t ns_consumed_count = 0;
    for (uint64_t i = 0; i < n_records; i++) {
        /* mate_chrom_id from MF. */
        if (mf[i] == MIV2_MF_NO_MATE) {
            out_mate_chrom_ids[i] = -1;
        } else if (mf[i] == MIV2_MF_SAME_CHROM) {
            int32_t own = (own_chrom_ids[i] == MIV2_OWN_UNMAPPED) ? -1 : (int32_t)own_chrom_ids[i];
            out_mate_chrom_ids[i] = own;
        } else {  /* CROSS_CHROM */
            uint64_t v;
            size_t consumed;
            int dr = miv2_varint_decode(ns_buf + ns_off, ns_decoded_size - ns_off, &v, &consumed);
            if (dr != 0) { free(mf); free(ns_buf); free(np_buf); free(ts_buf); return dr; }
            ns_off += consumed;
            ns_consumed_count++;
            out_mate_chrom_ids[i] = (int32_t)v;
        }

        /* mate_pos from NP. */
        uint64_t v;
        size_t consumed;
        int dr = miv2_varint_decode(np_buf + np_off, np_decoded_size - np_off, &v, &consumed);
        if (dr != 0) { free(mf); free(ns_buf); free(np_buf); free(ts_buf); return dr; }
        np_off += consumed;
        int64_t np_signed = miv2_zigzag_decode_64(v);
        if (mf[i] == MIV2_MF_SAME_CHROM) {
            out_mate_positions[i] = own_positions[i] + np_signed;
        } else {
            out_mate_positions[i] = np_signed;
        }

        /* tlen from TS. */
        dr = miv2_varint_decode(ts_buf + ts_off, ts_decoded_size - ts_off, &v, &consumed);
        if (dr != 0) { free(mf); free(ns_buf); free(np_buf); free(ts_buf); return dr; }
        ts_off += consumed;
        out_template_lengths[i] = (int32_t)miv2_zigzag_decode_64(v);
    }

    /* I4: NS length conservation check. */
    if (ns_consumed_count != num_cross || ns_off != ns_decoded_size) {
        free(mf); free(ns_buf); free(np_buf); free(ts_buf);
        return TTIO_RANS_ERR_NS_LENGTH_MISMATCH;
    }

    free(mf); free(ns_buf); free(np_buf); free(ts_buf);
    return 0;
}
```

```c
#include <assert.h>
#include <stdint.h>
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

#include "ttio_rans.h"
#include "mate_info_v2.h"

typedef enum {
    PATTERN_ALL_PROPER,
    PATTERN_ALL_CROSS,
    PATTERN_ALL_UNMAPPED,
    PATTERN_MIXED,
} pattern_t;

static void gen_record(pattern_t pat, uint64_t i, uint16_t *oc, int64_t *op,
                       int32_t *mc, int64_t *mp, int32_t *ts) {
    *oc = (uint16_t)(i % 24);
    *op = (int64_t)(i * 137);
    *ts = (int32_t)((i % 1000) - 500);
    switch (pat) {
        case PATTERN_ALL_PROPER:
            *mc = (int32_t)*oc;
            *mp = *op + ((int64_t)((i % 1000) - 500));
            break;
        case PATTERN_ALL_CROSS:
            *mc = (int32_t)((*oc + 1) % 24);
            *mp = (int64_t)((i * 311) % 100000000);
            break;
        case PATTERN_ALL_UNMAPPED:
            *mc = -1;
            *mp = 0;
            break;
        case PATTERN_MIXED:
        default: {
            uint64_t r = i % 10;
            if (r < 5) {  /* 50% same */
                *mc = (int32_t)*oc;
                *mp = *op + (int64_t)((i % 1000) - 500);
            } else if (r < 8) {  /* 30% cross */
                *mc = (int32_t)((*oc + 1) % 24);
                *mp = (int64_t)((i * 311) % 100000000);
            } else {  /* 20% unmapped */
                *mc = -1;
                *mp = 0;
            }
            break;
        }
    }
}

static void run_one(uint64_t n, pattern_t pat, const char *pat_name) {
    int32_t  *mc = malloc(n * sizeof(int32_t));
    int64_t  *mp = malloc(n * sizeof(int64_t));
    int32_t  *ts = malloc(n * sizeof(int32_t));
    uint16_t *oc = malloc(n * sizeof(uint16_t));
    int64_t  *op = malloc(n * sizeof(int64_t));
    for (uint64_t i = 0; i < n; i++) {
        gen_record(pat, i, &oc[i], &op[i], &mc[i], &mp[i], &ts[i]);
    }

    size_t cap = ttio_mate_info_v2_max_encoded_size(n);
    uint8_t *enc = malloc(cap);
    size_t enc_size = cap;
    int rc = ttio_mate_info_v2_encode(mc, mp, ts, oc, op, n, enc, &enc_size);
    assert(rc == 0);

    int32_t *mc2 = malloc(n * sizeof(int32_t));
    int64_t *mp2 = malloc(n * sizeof(int64_t));
    int32_t *ts2 = malloc(n * sizeof(int32_t));
    rc = ttio_mate_info_v2_decode(enc, enc_size, oc, op, n, mc2, mp2, ts2);
    assert(rc == 0);
    for (uint64_t i = 0; i < n; i++) {
        assert(mc[i] == mc2[i]);
        assert(mp[i] == mp2[i]);
        assert(ts[i] == ts2[i]);
    }
    printf("  n=%llu pattern=%-12s enc=%zu bytes (%.2f B/rec): PASS\n",

    free(mc2); free(mp2); free(ts2);

/* I4: NS length conservation — corrupt NS_LEN by ±1 and assert decoder
 * rejects with TTIO_RANS_ERR_NS_LENGTH_MISMATCH or TTIO_RANS_ERR_CORRUPT. */
static void test_i4_ns_length_conservation(void) {
    const uint64_t N = 100;
    int32_t  *mc = malloc(N * sizeof(int32_t));
    int64_t  *mp = malloc(N * sizeof(int64_t));
    int32_t  *ts = malloc(N * sizeof(int32_t));
    uint16_t *oc = malloc(N * sizeof(uint16_t));
    int64_t  *op = malloc(N * sizeof(int64_t));
    for (uint64_t i = 0; i < N; i++) {
        gen_record(PATTERN_ALL_CROSS, i, &oc[i], &op[i], &mc[i], &mp[i], &ts[i]);
    }
    size_t cap = ttio_mate_info_v2_max_encoded_size(N);
    uint8_t *enc = malloc(cap);
    size_t enc_size = cap;
    int rc = ttio_mate_info_v2_encode(mc, mp, ts, oc, op, N, enc, &enc_size);
    assert(rc == 0);

    /* Patch NUM_CROSS field (header offset 14) to N-1; decoder should
     * detect mismatch on NS consumption count or NP varint walk. */
    enc[14] = (uint8_t)((N - 1) & 0xff);
    int32_t mc2[100]; int64_t mp2[100]; int32_t ts2[100];
    rc = ttio_mate_info_v2_decode(enc, enc_size, oc, op, N, mc2, mp2, ts2);
    assert(rc != 0);
    printf("I4 NS length conservation: PASS (rc=%d on tampered NUM_CROSS)\n", rc);

    free(mc); free(mp); free(ts); free(oc); free(op); free(enc);
}

/* I5: MF auto-pick equivalence — synthesize MF[] arrays where rANS will
 * win vs raw-pack and vice versa; round-trip both and assert equality. */
static void test_i5_mf_autopick_equivalence(void) {
    /* Case A: 1000 records, all MF=0 (highly compressible by rANS). */
    /* Case B: 8 records, alternating MF (raw-pack wins on tiny N). */
    uint64_t cases[][2] = {{1000, 0}, {8, 1}};
    for (int c = 0; c < 2; c++) {
        uint64_t n = cases[c][0];
        int32_t  *mc = malloc(n * sizeof(int32_t));
        int64_t  *mp = malloc(n * sizeof(int64_t));
        int32_t  *ts = malloc(n * sizeof(int32_t));
        uint16_t *oc = malloc(n * sizeof(uint16_t));
        int64_t  *op = malloc(n * sizeof(int64_t));
        for (uint64_t i = 0; i < n; i++) {
            oc[i] = (uint16_t)(i % 24);
            op[i] = (int64_t)(i * 137);
            ts[i] = (int32_t)i;
            if (c == 0) {
                mc[i] = (int32_t)oc[i];  /* all SAME */
                mp[i] = op[i];
            } else {
                /* Alternating across all 3 MF values */
                int v = (int)(i % 3);
                if (v == 0) { mc[i] = (int32_t)oc[i]; mp[i] = op[i]; }
                else if (v == 1) { mc[i] = (int32_t)((oc[i]+1)%24); mp[i] = i*7; }
                else { mc[i] = -1; mp[i] = 0; }
            }
        }

        size_t cap = ttio_mate_info_v2_max_encoded_size(n);
        uint8_t *enc = malloc(cap);
        size_t enc_size = cap;
        int rc = ttio_mate_info_v2_encode(mc, mp, ts, oc, op, n, enc, &enc_size);
        assert(rc == 0);

        int32_t *mc2 = malloc(n * sizeof(int32_t));
        int64_t *mp2 = malloc(n * sizeof(int64_t));
        int32_t *ts2 = malloc(n * sizeof(int32_t));
        rc = ttio_mate_info_v2_decode(enc, enc_size, oc, op, n, mc2, mp2, ts2);
        assert(rc == 0);
        for (uint64_t i = 0; i < n; i++) {
            assert(mc[i] == mc2[i]);
            assert(mp[i] == mp2[i]);
            assert(ts[i] == ts2[i]);
        }
        free(mc); free(mp); free(ts); free(oc); free(op); free(enc);
        free(mc2); free(mp2); free(ts2);
    }
    printf("I5 MF auto-pick equivalence: PASS (both small + large cases)\n");
}
```

```python
# python/src/ttio/codecs/mate_info_v2.py
"""Python ctypes wrapper for mate_info v2 (CRAM-style inline mate-pair codec).

Spec: docs/superpowers/specs/2026-05-03-mate-info-v2-design.md

Encoded blob written as ``signal_channels/mate_info/inline_v2`` uint8
dataset with ``@compression = 13`` (MATE_INLINE_V2). Requires
``TTIO_RANS_LIB_PATH`` to point at a built ``libttio_rans.so``;
without it, ``HAVE_NATIVE_LIB`` is False and v2 write/read is
disabled (use ``opt_disable_inline_mate_info_v2 = True`` to fall back
to v1).
"""
from __future__ import annotations

import ctypes
import os

import numpy as np

# Reuse the loader from fqzcomp_nx16_z.py — same library, same env var.
from .fqzcomp_nx16_z import _native_lib, _HAVE_NATIVE_LIB

HAVE_NATIVE_LIB: bool = _HAVE_NATIVE_LIB

# Error codes from native/include/ttio_rans.h
ERR_PARAM = -1
ERR_ALLOC = -2
ERR_CORRUPT = -3
ERR_RESERVED_MF = -4
ERR_NS_LENGTH_MISMATCH = -5

_ERR_MESSAGES = {
    ERR_PARAM:    "invalid parameters",
    ERR_ALLOC:    "out of memory in native code",
    ERR_CORRUPT:  "corrupt encoded blob",
    ERR_RESERVED_MF: "reserved MF value 3 in stream",
    ERR_NS_LENGTH_MISMATCH: "NS substream length does not match NUM_CROSS",
}


if HAVE_NATIVE_LIB:
    _lib = _native_lib

    _lib.ttio_mate_info_v2_max_encoded_size.argtypes = [ctypes.c_uint64]
    _lib.ttio_mate_info_v2_max_encoded_size.restype = ctypes.c_size_t

    _lib.ttio_mate_info_v2_encode.argtypes = [
        ctypes.POINTER(ctypes.c_int32),   # mate_chrom_ids
        ctypes.POINTER(ctypes.c_int64),   # mate_positions
        ctypes.POINTER(ctypes.c_int32),   # template_lengths
        ctypes.POINTER(ctypes.c_uint16),  # own_chrom_ids
        ctypes.POINTER(ctypes.c_int64),   # own_positions
        ctypes.c_uint64,                  # n_records
        ctypes.POINTER(ctypes.c_uint8),   # out
        ctypes.POINTER(ctypes.c_size_t),  # out_len
    ]
    _lib.ttio_mate_info_v2_encode.restype = ctypes.c_int

    _lib.ttio_mate_info_v2_decode.argtypes = [
        ctypes.POINTER(ctypes.c_uint8),   # encoded
        ctypes.c_size_t,                  # encoded_size
        ctypes.POINTER(ctypes.c_uint16),  # own_chrom_ids
        ctypes.POINTER(ctypes.c_int64),   # own_positions
        ctypes.c_uint64,                  # n_records
        ctypes.POINTER(ctypes.c_int32),   # out_mate_chrom_ids
        ctypes.POINTER(ctypes.c_int64),   # out_mate_positions
        ctypes.POINTER(ctypes.c_int32),   # out_template_lengths
    ]
    _lib.ttio_mate_info_v2_decode.restype = ctypes.c_int


def _check_input_validity(mate_chrom_ids):
    """Mirror the native miv2_classify_mf input validation."""
    if np.any(mate_chrom_ids < -1):
        bad = mate_chrom_ids[mate_chrom_ids < -1][0]
        raise ValueError(
            f"invalid mate_chrom_id: {bad} (must be >= -1)")


def encode(
    mate_chrom_ids: np.ndarray,
    mate_positions: np.ndarray,
    template_lengths: np.ndarray,
    own_chrom_ids: np.ndarray,
    own_positions: np.ndarray,
) -> bytes:
    """Encode a mate triple to the inline_v2 blob.

    All arrays must be 1-D and have the same length. Dtypes are
    enforced via ``np.ascontiguousarray``.
    """
    if not HAVE_NATIVE_LIB:
        raise RuntimeError(
            "mate_info_v2.encode requires libttio_rans (set "
            "TTIO_RANS_LIB_PATH); set opt_disable_inline_mate_info_v2 "
            "= True on WrittenGenomicRun to fall back to v1.")

    mc = np.ascontiguousarray(mate_chrom_ids, dtype=np.int32)
    mp = np.ascontiguousarray(mate_positions, dtype=np.int64)
    ts = np.ascontiguousarray(template_lengths, dtype=np.int32)
    oc = np.ascontiguousarray(own_chrom_ids, dtype=np.uint16)
    op = np.ascontiguousarray(own_positions, dtype=np.int64)

    n = mc.shape[0]
    if not (mp.shape[0] == n and ts.shape[0] == n
            and oc.shape[0] == n and op.shape[0] == n):
        raise ValueError("all input arrays must have the same length")

    _check_input_validity(mc)

    if n == 0:
        # Empty input: emit an empty-record header.
        cap = _lib.ttio_mate_info_v2_max_encoded_size(0)
        out = (ctypes.c_uint8 * cap)()
        out_len = ctypes.c_size_t(cap)
        rc = _lib.ttio_mate_info_v2_encode(
            mc.ctypes.data_as(ctypes.POINTER(ctypes.c_int32)),
            mp.ctypes.data_as(ctypes.POINTER(ctypes.c_int64)),
            ts.ctypes.data_as(ctypes.POINTER(ctypes.c_int32)),
            oc.ctypes.data_as(ctypes.POINTER(ctypes.c_uint16)),
            op.ctypes.data_as(ctypes.POINTER(ctypes.c_int64)),
            ctypes.c_uint64(0),
            out, ctypes.byref(out_len))
        if rc != 0:
            raise RuntimeError(_ERR_MESSAGES.get(rc, f"native error {rc}"))
        return bytes(out[:out_len.value])

    cap = _lib.ttio_mate_info_v2_max_encoded_size(n)
    out = (ctypes.c_uint8 * cap)()
    out_len = ctypes.c_size_t(cap)
    rc = _lib.ttio_mate_info_v2_encode(
        mc.ctypes.data_as(ctypes.POINTER(ctypes.c_int32)),
        mp.ctypes.data_as(ctypes.POINTER(ctypes.c_int64)),
        ts.ctypes.data_as(ctypes.POINTER(ctypes.c_int32)),
        oc.ctypes.data_as(ctypes.POINTER(ctypes.c_uint16)),
        op.ctypes.data_as(ctypes.POINTER(ctypes.c_int64)),
        ctypes.c_uint64(n),
        out, ctypes.byref(out_len))
    if rc != 0:
        raise RuntimeError(_ERR_MESSAGES.get(rc, f"native error {rc}"))
    return bytes(out[:out_len.value])


def decode(
    encoded: bytes,
    own_chrom_ids: np.ndarray,
    own_positions: np.ndarray,
    n_records: int,
) -> tuple[np.ndarray, np.ndarray, np.ndarray]:
    """Decode an inline_v2 blob to (mate_chrom_ids, mate_positions, template_lengths)."""
    if not HAVE_NATIVE_LIB:
        raise RuntimeError(
            "mate_info_v2.decode requires libttio_rans (set TTIO_RANS_LIB_PATH)")

    oc = np.ascontiguousarray(own_chrom_ids, dtype=np.uint16)
    op = np.ascontiguousarray(own_positions, dtype=np.int64)
    if oc.shape[0] != n_records or op.shape[0] != n_records:
        raise ValueError(
            f"own_chrom_ids/own_positions length must equal n_records ({n_records})")

    enc_arr = np.frombuffer(encoded, dtype=np.uint8)
    out_mc = np.empty(n_records, dtype=np.int32)
    out_mp = np.empty(n_records, dtype=np.int64)
    out_ts = np.empty(n_records, dtype=np.int32)

    rc = _lib.ttio_mate_info_v2_decode(
        enc_arr.ctypes.data_as(ctypes.POINTER(ctypes.c_uint8)),
        ctypes.c_size_t(len(encoded)),
        oc.ctypes.data_as(ctypes.POINTER(ctypes.c_uint16)),
        op.ctypes.data_as(ctypes.POINTER(ctypes.c_int64)),
        ctypes.c_uint64(n_records),
        out_mc.ctypes.data_as(ctypes.POINTER(ctypes.c_int32)),
        out_mp.ctypes.data_as(ctypes.POINTER(ctypes.c_int64)),
        out_ts.ctypes.data_as(ctypes.POINTER(ctypes.c_int32)))
    if rc != 0:
        raise RuntimeError(_ERR_MESSAGES.get(rc, f"native error {rc}"))
    return out_mc, out_mp, out_ts
```

```python
# python/tests/integration/_mate_info_corpus.py
"""Extract mate-info triples from a BAM file for v1↔v2 oracle testing.

Returns the same parallel-array shape used by mate_info_v2.encode():
(mate_chrom_ids, mate_positions, template_lengths,
 own_chrom_ids, own_positions).
"""
from __future__ import annotations

from pathlib import Path

import numpy as np
import pysam


def extract_mate_triples(bam_path: Path):
    """Return parallel int32/int64/int32/uint16/int64 arrays.

    Encounter-order chrom_id assignment matches the L1 contract.
    """
    chrom_id_map: dict[str, int] = {}

    def _id_for(name: str) -> int:
        if name not in chrom_id_map:
            chrom_id_map[name] = len(chrom_id_map)
        return chrom_id_map[name]

    own_chrom_ids = []
    own_positions = []
    mate_chrom_ids = []
    mate_positions = []
    template_lengths = []

    with pysam.AlignmentFile(str(bam_path), "rb") as bam:
        for r in bam.fetch(until_eof=True):
            if r.is_unmapped:
                own_chrom_ids.append(0xFFFF)
                own_positions.append(0)
            else:
                own_chrom_ids.append(_id_for(r.reference_name))
                own_positions.append(r.reference_start)

            if r.next_reference_id < 0 or r.next_reference_name is None:
                mate_chrom_ids.append(-1)
                mate_positions.append(0)
            else:
                mate_chrom_ids.append(_id_for(r.next_reference_name))
                mate_positions.append(r.next_reference_start)

            template_lengths.append(r.template_length)

    return (
        np.asarray(mate_chrom_ids, dtype=np.int32),
        np.asarray(mate_positions, dtype=np.int64),
        np.asarray(template_lengths, dtype=np.int32),
        np.asarray(own_chrom_ids, dtype=np.uint16),
        np.asarray(own_positions, dtype=np.int64),
    )
```

```java
// java/src/main/java/global/thalion/ttio/codecs/MateInfoV2.java
package global.thalion.ttio.codecs;

/**
 * Java JNI binding for mate_info v2 (CRAM-style inline mate-pair codec).
 *
 * Spec: docs/superpowers/specs/2026-05-03-mate-info-v2-design.md
 *
 * Wraps the C library entries ttio_mate_info_v2_encode / _decode.
 * Loads {@code libttio_rans_jni.so} via {@link TtioRansNative} (which
 * also handles V4 and rANS bindings — single shared library).
 */
public final class MateInfoV2 {

    static {
        // Force the same loader as TtioRansNative (System.loadLibrary("ttio_rans_jni")).
        TtioRansNative.ensureLoaded();
    }

    private MateInfoV2() {}

    /**
     * Encode a mate triple to the inline_v2 blob.
     *
     * @param mateChromIds    int[] — per record, -1 if RNEXT='*'
     * @param matePositions   long[] — per record, 0-based POS
     * @param templateLengths int[] — per record, signed tlen
     * @param ownChromIds     char[] — per record, uint16 from L1 chrom_ids; 0xFFFF = unmapped
     * @param ownPositions    long[] — per record, from genomic_index/positions
     * @return encoded blob ready for HDF5 write
     * @throws RuntimeException if input is invalid or native call fails
     */
    public static byte[] encode(
            int[]  mateChromIds,
            long[] matePositions,
            int[]  templateLengths,
            char[] ownChromIds,
            long[] ownPositions) {
        int n = mateChromIds.length;
        if (matePositions.length != n || templateLengths.length != n
                || ownChromIds.length != n || ownPositions.length != n) {
            throw new IllegalArgumentException(
                "all input arrays must have the same length");
        }
        for (int i = 0; i < n; i++) {
            if (mateChromIds[i] < -1) {
                throw new IllegalArgumentException(
                    "invalid mate_chrom_id at index " + i + ": "
                    + mateChromIds[i] + " (must be >= -1)");
            }
        }
        return encodeNative(mateChromIds, matePositions, templateLengths,
                            ownChromIds, ownPositions);
    }

    /** Decode an inline_v2 blob to (mate_chrom_ids, mate_positions, template_lengths). */
    public static Triple decode(
            byte[] encoded,
            char[] ownChromIds,
            long[] ownPositions,
            int    nRecords) {
        if (ownChromIds.length != nRecords || ownPositions.length != nRecords) {
            throw new IllegalArgumentException(
                "ownChromIds/ownPositions length must equal nRecords");
        }
        int[]  mc = new int[nRecords];
        long[] mp = new long[nRecords];
        int[]  ts = new int[nRecords];
        decodeNative(encoded, ownChromIds, ownPositions, mc, mp, ts);
        return new Triple(mc, mp, ts);
    }

    public static final class Triple {
        public final int[]  mateChromIds;
        public final long[] matePositions;
        public final int[]  templateLengths;
        public Triple(int[] mc, long[] mp, int[] ts) {
            this.mateChromIds = mc;
            this.matePositions = mp;
            this.templateLengths = ts;
        }
    }

    private static native byte[] encodeNative(
        int[] mateChromIds, long[] matePositions, int[] templateLengths,
        char[] ownChromIds, long[] ownPositions);

    private static native void decodeNative(
        byte[] encoded, char[] ownChromIds, long[] ownPositions,
        int[] outMateChromIds, long[] outMatePositions, int[] outTemplateLengths);
}
```

```java
import global.thalion.ttio.codecs.MateInfoV2;
import htsjdk.samtools.SAMRecord;
import htsjdk.samtools.SamReader;
import htsjdk.samtools.SamReaderFactory;
import htsjdk.samtools.ValidationStringency;

import java.io.File;
import java.io.IOException;
import java.util.ArrayList;
import java.util.HashMap;
import java.util.List;
import java.util.Map;

/**
 * CLI tool that reads a BAM, extracts mate triples, encodes via
 * MateInfoV2.encode, writes the encoded blob to stdout.
 *
 * Usage: MateInfoV2Cli <bam_path>
 *
 * Used by the cross-language byte-exact gate (Task 11) to compare
 * encoder output across Python / Java / ObjC for the same input.
 */
public final class MateInfoV2Cli {

    public static void main(String[] args) throws IOException {
        if (args.length != 1) {
            System.err.println("Usage: MateInfoV2Cli <bam_path>");
            System.exit(2);
        }
        File bam = new File(args[0]);

        Map<String, Integer> chromIdMap = new HashMap<>();
        List<Integer> mc = new ArrayList<>();
        List<Long>    mp = new ArrayList<>();
        List<Integer> ts = new ArrayList<>();
        List<Character> oc = new ArrayList<>();
        List<Long>    op = new ArrayList<>();

        SamReaderFactory factory = SamReaderFactory.makeDefault()
            .validationStringency(ValidationStringency.SILENT);
        try (SamReader reader = factory.open(bam)) {
            for (SAMRecord r : reader) {
                if (r.getReadUnmappedFlag()) {
                    oc.add((char) 0xFFFF);
                    op.add(0L);
                } else {
                    String name = r.getReferenceName();
                    Integer id = chromIdMap.get(name);
                    if (id == null) {
                        id = chromIdMap.size();
                        chromIdMap.put(name, id);
                    }
                    oc.add((char) (int) id);
                    op.add((long) r.getAlignmentStart() - 1);  /* 1-based -> 0-based */
                }

                int mateRef = r.getMateReferenceIndex();
                if (mateRef < 0 || r.getMateReferenceName() == null
                        || r.getMateReferenceName().equals("*")) {
                    mc.add(-1);
                    mp.add(0L);
                } else {
                    String name = r.getMateReferenceName();
                    Integer id = chromIdMap.get(name);
                    if (id == null) {
                        id = chromIdMap.size();
                        chromIdMap.put(name, id);
                    }
                    mc.add(id);
                    mp.add((long) r.getMateAlignmentStart() - 1);
                }
                ts.add(r.getInferredInsertSize());
            }
        }

        int n = mc.size();
        int[]  mcA = new int[n];
        long[] mpA = new long[n];
        int[]  tsA = new int[n];
        char[] ocA = new char[n];
        long[] opA = new long[n];
        for (int i = 0; i < n; i++) {
            mcA[i] = mc.get(i);
            mpA[i] = mp.get(i);
            tsA[i] = ts.get(i);
            ocA[i] = oc.get(i);
            opA[i] = op.get(i);
        }

        byte[] encoded = MateInfoV2.encode(mcA, mpA, tsA, ocA, opA);
        System.out.write(encoded);
        System.out.flush();
    }
}
```

```objc
NS_ASSUME_NONNULL_BEGIN

/**
 * mate_info v2 — CRAM-style inline mate-pair codec.
 *
 * Spec: docs/superpowers/specs/2026-05-03-mate-info-v2-design.md
 *
 * Direct link to the C library entries ttio_mate_info_v2_encode /
 * _decode in libttio_rans (header at <ttio_rans.h>).
 */
@interface TTIOMateInfoV2 : NSObject

/** Encode a mate triple to the inline_v2 blob. All NSData inputs
 *  are interpreted as parallel arrays of the noted typed elements
 *  (int32, int64, int32, uint16, int64). Returns nil + error on
 *  invalid input or native error. */
+ (nullable NSData *)encodeMateChromIds:(NSData *)mateChromIds
                          matePositions:(NSData *)matePositions
                        templateLengths:(NSData *)templateLengths
                            ownChromIds:(NSData *)ownChromIds
                           ownPositions:(NSData *)ownPositions
                                  error:(NSError **)error;

/** Decode an inline_v2 blob to (mate_chrom_ids, mate_positions,
 *  template_lengths). Returns YES on success, NO + error on failure. */
+ (BOOL)decodeData:(NSData *)encoded
        ownChromIds:(NSData *)ownChromIds
       ownPositions:(NSData *)ownPositions
           nRecords:(NSUInteger)nRecords
```

```objc
// objc/Tools/TtioMateInfoV2Cli.m
//
// Reads a BAM via samtools (subprocess), extracts mate triples in
// encounter-order chrom-id form, encodes via TTIOMateInfoV2, writes
// the encoded blob to stdout.
//
// Usage: TtioMateInfoV2Cli <bam_path>
//
// The BAM extraction is delegated to `samtools view` to avoid a
// hard htslib dependency (per TTIOBamReader.h: "no htslib link"
// project policy).
#import <Foundation/Foundation.h>
#import "TTIOMateInfoV2.h"

#import <stdio.h>
#import <stdlib.h>
#import <string.h>

static int parse_record_line(const char *line, int *flag, int *pos,
                             const char **rname, const char **mrname,
                             int *mpos, int *tlen) {
    /* SAM fields tab-separated: QNAME FLAG RNAME POS MAPQ CIGAR
     * RNEXT PNEXT TLEN SEQ QUAL ... */
    static char buf[65536];
    static const char *fields[16];
    strncpy(buf, line, sizeof(buf));
    buf[sizeof(buf)-1] = '\0';
    int n = 0;
    char *p = buf;
    fields[n++] = p;
    while (*p && n < 16) {
        if (*p == '\t') { *p = '\0'; fields[n++] = p+1; }
        p++;
    }
    if (n < 9) return -1;
    *flag   = atoi(fields[1]);
    *rname  = fields[2];
    *pos    = atoi(fields[3]);  /* 1-based */
    *mrname = fields[6];
    *mpos   = atoi(fields[7]);  /* 1-based */
    *tlen   = atoi(fields[8]);
    return 0;
}

int main(int argc, const char *argv[]) {
    @autoreleasepool {
        if (argc != 2) {
            fprintf(stderr, "Usage: TtioMateInfoV2Cli <bam_path>\n");
            return 2;
        }
        char cmd[2048];
        snprintf(cmd, sizeof(cmd), "samtools view %s", argv[1]);
        FILE *fp = popen(cmd, "r");
        if (!fp) { perror("popen"); return 1; }

        NSMutableDictionary<NSString *, NSNumber *> *chromMap = [NSMutableDictionary dictionary];
        NSMutableData *mc = [NSMutableData data];
        NSMutableData *mp = [NSMutableData data];
        NSMutableData *ts = [NSMutableData data];
        NSMutableData *oc = [NSMutableData data];
        NSMutableData *op = [NSMutableData data];

        char line[65536];
        while (fgets(line, sizeof(line), fp)) {
            int flag, pos, mpos, tlen;
            const char *rname, *mrname;
            if (parse_record_line(line, &flag, &pos, &rname, &mrname, &mpos, &tlen) != 0) continue;

            uint16_t own_chrom_id;
            int64_t  own_pos;
            if (flag & 0x4) {  /* unmapped */
                own_chrom_id = 0xFFFFu;
                own_pos = 0;
            } else {
                NSString *key = [NSString stringWithUTF8String:rname];
                NSNumber *id = chromMap[key];
                if (!id) {
                    id = @([chromMap count]);
                    chromMap[key] = id;
                }
                own_chrom_id = (uint16_t)[id intValue];
                own_pos = (int64_t)pos - 1;  /* 1-based -> 0-based */
            }

            int32_t mate_chrom_id;
            int64_t mate_pos;
            if (strcmp(mrname, "*") == 0) {
                mate_chrom_id = -1;
                mate_pos = 0;
            } else {
                const char *mname = (strcmp(mrname, "=") == 0) ? rname : mrname;
                NSString *key = [NSString stringWithUTF8String:mname];
                NSNumber *id = chromMap[key];
                if (!id) {
                    id = @([chromMap count]);
                    chromMap[key] = id;
                }
                mate_chrom_id = (int32_t)[id intValue];
                mate_pos = (int64_t)mpos - 1;
            }
            int32_t tlen_v = (int32_t)tlen;

            [mc appendBytes:&mate_chrom_id length:sizeof(mate_chrom_id)];
            [mp appendBytes:&mate_pos length:sizeof(mate_pos)];
            [ts appendBytes:&tlen_v length:sizeof(tlen_v)];
            [oc appendBytes:&own_chrom_id length:sizeof(own_chrom_id)];
            [op appendBytes:&own_pos length:sizeof(own_pos)];
        }
        pclose(fp);

        NSError *err = nil;
        NSData *encoded = [TTIOMateInfoV2 encodeMateChromIds:mc
                                             matePositions:mp
                                           templateLengths:ts
                                              ownChromIds:oc
                                             ownPositions:op
                                                    error:&err];
        if (!encoded) {
            fprintf(stderr, "encode failed: %s\n", [[err localizedDescription] UTF8String]);
            return 1;
        }
        fwrite([encoded bytes], 1, [encoded length], stdout);
    }
    return 0;
}
```

```python
def _write_mate_info_inline_v2(sc, run):
    """v1.7 inline_v2 writer: encode the full mate triple via libttio_rans
    and write as a single uint8 blob at signal_channels/mate_info/inline_v2
    with @compression = 13."""
    from .codecs import mate_info_v2 as _miv2
    from .enums import Compression as _Compression, Precision as _Precision

    mate_group = sc.create_group("mate_info")

    # Build the parallel typed arrays. mate_chromosomes is a list[str];
    # we need int32 chrom_ids matching genomic_index/chromosome_ids.
    own_chrom_ids, _ = _build_chrom_id_table(run.chromosomes)
    mate_chrom_ids = _resolve_mate_chrom_ids(run.mate_chromosomes, own_chrom_ids, run.chromosomes)

    encoded = _miv2.encode(
        mate_chrom_ids=mate_chrom_ids,
        mate_positions=np.asarray(run.mate_positions, dtype=np.int64),
        template_lengths=np.asarray(run.template_lengths, dtype=np.int32),
        own_chrom_ids=own_chrom_ids.astype(np.uint16, copy=False),
        own_positions=np.asarray(run.positions, dtype=np.int64),
    )
    arr = np.frombuffer(encoded, dtype=np.uint8)
    ds = mate_group.create_dataset(
        "inline_v2",
        _Precision.UINT8,
        length=int(arr.shape[0]),
        chunk_size=io.DEFAULT_SIGNAL_CHUNK,
        compression=_Compression.NONE,
    )
    ds.write(arr)
    io.write_int_attr(ds, "compression", int(_Compression.MATE_INLINE_V2), dtype="<u1")


def _build_chrom_id_table(chromosomes):
    """Encounter-order chrom_id assignment matching the L1 contract.

    Returns (uint16 array of chrom_ids per record, dict name->id).
    Use 0xFFFF for unmapped records ('*' or empty).
    """
    name_to_id: dict[str, int] = {}
    ids = np.empty(len(chromosomes), dtype=np.uint16)
    for i, name in enumerate(chromosomes):
        if name == "*" or not name:
            ids[i] = 0xFFFF
            continue
        if name not in name_to_id:
            name_to_id[name] = len(name_to_id)
        ids[i] = name_to_id[name]
    return ids, name_to_id


def _resolve_mate_chrom_ids(mate_chromosomes, own_chrom_ids, chromosomes):
    """Map mate chromosome names to int32 ids; -1 for '*'.

    Uses the same encounter-order dict as own_chrom_ids so SAME_CHROM
    classification works without a separate dictionary.
    """
    # Rebuild the same name->id dict (in record order over chromosomes).
    name_to_id: dict[str, int] = {}
    for name in chromosomes:
        if name == "*" or not name:
            continue
        if name not in name_to_id:
            name_to_id[name] = len(name_to_id)

    n = len(mate_chromosomes)
    out = np.empty(n, dtype=np.int32)
    for i, name in enumerate(mate_chromosomes):
        if name == "*" or not name:
            out[i] = -1
        elif name == "=":
            # Resolve to own chrom_id (already encoded).
            out[i] = -1 if own_chrom_ids[i] == 0xFFFF else int(own_chrom_ids[i])
        else:
            if name not in name_to_id:
                # Mate's chrom never appears as own chrom; extend dict.
                name_to_id[name] = len(name_to_id)
            out[i] = name_to_id[name]
    return out
```

```markdown
## 10.9b mate_info v2 inline codec (M11 v1.7, codec id 13)

v1.7 introduces a single-blob inline encoding of the full mate
triple (`mate_chrom_id`, `mate_pos`, `tlen`) that exploits SAM
mate-pair invariants. Default ON in v1.7; disable via
`WrittenGenomicRun.opt_disable_inline_mate_info_v2 = True` to fall
back to the v1 layout from §10.9.

### 10.9b.1 On-disk schema

Single uint8 dataset at `signal_channels/mate_info/inline_v2`,
with `@compression = 13` (`MATE_INLINE_V2`). The container
includes a 34-byte header followed by 4 substreams (MF / NS / NP /
TS). Full wire format spec: `docs/superpowers/specs/2026-05-03-mate-info-v2-design.md` §4.

### 10.9b.2 Reader-side dependency

Decoding `inline_v2` requires `genomic_index/positions` and
`genomic_index/chromosome_ids` to be loaded first. The decoder
needs `own_pos` and `own_chrom_id` per record to reconstruct
`mate_pos` for SAME_CHROM records (which use a delta encoding)
and to validate the MF taxonomy.

### 10.9b.3 Backward compatibility

A v1.6 reader on a v1.7 file fails on `inline_v2` with "unknown
compression id 13"; the user must upgrade or write the file with
`opt_disable_inline_mate_info_v2 = True`.

A v1.7 reader on a v1.6 file finds no `inline_v2` dataset and
falls through to the v1 layout from §10.9 transparently.

### 10.9b.4 signal_codec_overrides interaction
```
