# MPEG-O — Java Implementation (planned)

Java implementation — planned. Will follow the Objective-C reference implementation architecture.

See [`../ARCHITECTURE.md`](../ARCHITECTURE.md) and [`../WORKPLAN.md`](../WORKPLAN.md) for the canonical class hierarchy and milestone plan. The Java stream will use the HDF Group's `jhdf5` bindings and mirror the `MPGO` class names without the prefix (e.g. `org.mpego.SignalArray`, `org.mpego.MassSpectrum`).
